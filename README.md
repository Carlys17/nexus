# Nexus Network

Zero-knowledge proof generation and verification using Nexus Network's parallel proving infrastructure.

## What is Nexus?

Nexus is a verifiable computing network that generates and verifies zero-knowledge proofs at scale.

## Features

- ⚡ Parallel proof generation
- 🔐 ZK verification
- 💻 CLI interface
- 🌐 Decentralized prover network

## Installation

```bash
# Install Nexus CLI
curl -sSL https://cli.nexus.xyz/ | sh
```

## Usage

```bash
# Start prover
nexus-cli start

# Generate proof for program
nexus-cli prove --program my_program
```
